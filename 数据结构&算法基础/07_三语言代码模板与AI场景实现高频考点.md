<h1 id="q-001">1. 哪些数据结构与算法问答需要配套代码？</h1>

数据结构与算法板块中，并不是每个问答都适合塞一大段代码。更合理的方式是先判断问题类型：

| 问题类型 | 是否需要代码 | 例子 | 面试要求 |
| --- | --- | --- | --- |
| 概念辨析 | 不一定 | 数组 vs 链表、稳定排序、ANN vs kNN | 讲清定义、复杂度、适用场景 |
| 模板算法 | 必须会 | 二分、DFS、BFS、拓扑排序、KMP、DP | 能手写正确代码 |
| 工程结构 | 建议会 | LRU、Top K、Trie、Bitmap、Bloom Filter | 能写核心实现并说明工程坑点 |
| 系统设计 | 不宜只写代码 | RAG、LLM 推理调度、Agent Workflow | 画结构、讲数据流和权衡 |
| 数值算法 | 建议会 | 牛顿迭代、高斯消元 | 能写简化版并说明数值稳定性 |

本篇文档的定位是：为 01-06 主线问答提供三语言代码模板。正文问答负责“为什么”，本篇负责“怎么写”。

<h2 id="q-002">面试问题：哪些问题只需要概念解释，哪些问题必须会手写代码？</h2>

必须会手写代码的问题：

- 链表中点、链表环检测。
- LRU Cache。
- 单调栈、单调队列。
- 归并排序、快速排序、堆 / Top K。
- 二分查找及左右边界。
- 二叉树 DFS / BFS。
- 图 BFS / DFS、二分图、拓扑排序、Dijkstra。
- KMP、Trie。
- 回溯、动态规划、滑动窗口。
- 牛顿迭代、基础计算几何。

更适合概念解释的问题：

- Tensor 与数组关系。
- HNSW、IVF、PQ 的工程原理。
- RAG 混合检索架构。
- KV Cache、Prefix Cache、Continuous Batching。
- Agent Workflow 的 DAG 建模。

原因很简单：系统设计题靠代码片段无法完整表达；而模板算法题如果只会讲概念，面试官很难判断你的基本功。

<h2 id="q-005">面试问题：如何用三种语言实现链表中点？</h2>

核心思想：快慢指针。慢指针每次走一步，快指针每次走两步，快指针到尾部时慢指针在中点。

AI 场景类比：流式缓冲区、token 序列截断和滑动窗口都可以借鉴“双指针不同速度扫描”的思想。

Python：

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def middle_node(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
    return slow
```

C/C++：

```cpp

    int val;
    ListNode* next;
    ListNode(int v) : val(v), next(nullptr) {}

    ListNode* fast = head;
    while (fast != nullptr && fast->next != nullptr) {
        slow = slow->next;
        fast = fast->next->next;
    }
    return slow;
```


<h2 id="q-006">面试问题：如何用三种语言实现 LRU Cache？</h2>

核心结构：哈希表 + 双向链表。

- 哈希表负责 $O(1)$ 定位节点。
- 双向链表负责 $O(1)$ 移动节点和淘汰尾部节点。

AI 场景：

- RAG query cache。
- embedding cache。
- session cache。
- prefix / KV cache 淘汰策略。

Python：

```python
class Node:
    def __init__(self, key=0, val=0):
        self.key = key
        self.val = val
        self.prev = None
        self.next = None

class LRUCache:
    def __init__(self, capacity):
        self.capacity = capacity
        self.cache = {}
        self.head = Node()
        self.tail = Node()
        self.head.next = self.tail
        self.tail.prev = self.head

    def _remove(self, node):
        node.prev.next = node.next
        node.next.prev = node.prev

    def _add_front(self, node):
        node.next = self.head.next
        node.prev = self.head
        self.head.next.prev = node
        self.head.next = node

    def get(self, key):
        if key not in self.cache:
            return -1
        node = self.cache[key]
        self._remove(node)
        self._add_front(node)
        return node.val

    def put(self, key, value):
        if key in self.cache:
            node = self.cache[key]
            node.val = value
            self._remove(node)
            self._add_front(node)
            return
        node = Node(key, value)
        self.cache[key] = node
        self._add_front(node)
        if len(self.cache) > self.capacity:
            old = self.tail.prev
            self._remove(old)
            del self.cache[old.key]
```

C/C++：

```cpp
#include <list>
#include <unordered_map>
using namespace std;

class LRUCache {
private:
    int cap;
    list<pair<int, int>> items;
    unordered_map<int, list<pair<int, int>>::iterator> pos;

public:
    LRUCache(int capacity) : cap(capacity) {}

    int get(int key) {
        if (!pos.count(key)) return -1;
        auto it = pos[key];
        int value = it->second;
        items.splice(items.begin(), items, it);
        return value;
    }

    void put(int key, int value) {
        if (pos.count(key)) {
            auto it = pos[key];
            it->second = value;
            items.splice(items.begin(), items, it);
            return;
        }
        items.push_front({key, value});
        pos[key] = items.begin();
        if ((int)items.size() > cap) {
            int oldKey = items.back().first;
            items.pop_back();
            pos.erase(oldKey);
        }
    }
};
```

Java：

```java

import java.util.Map;

        super.put(key, value);
    }
```

<h2 id="q-007">面试问题：如何用三种语言实现单调栈和滑动窗口最大值？</h2>

单调栈适合找下一个更大 / 更小元素；单调队列适合滑动窗口最大值。

AI 场景：

- 流式指标窗口最大延迟。
- 最近 N 个 token 或请求中的最大风险分。
- 时间序列异常检测中的窗口统计。

Python：

```python
from collections import deque

def next_greater(nums):
    ans = [-1] * len(nums)
    st = []
    for i, x in enumerate(nums):
        while st and nums[st[-1]] < x:
            ans[st.pop()] = x
        st.append(i)
    return ans

def max_sliding_window(nums, k):
    q = deque()
    ans = []
    for i, x in enumerate(nums):
        while q and q[0] <= i - k:
            q.popleft()
        while q and nums[q[-1]] <= x:
            q.pop()
        q.append(i)
        if i >= k - 1:
            ans.append(nums[q[0]])
    return ans
```

C/C++：

```cpp
#include <deque>

        while (!q.empty() && nums[q.back()] <= nums[i]) q.pop_back();
        q.push_back(i);
        if (i >= k - 1) ans.push_back(nums[q.front()]);
    }
    return ans;
}
```

Java：

```java
import java.util.*;

class MonoTemplates {
    int[] nextGreater(int[] nums) {
        int[] ans = new int[nums.length];
        Arrays.fill(ans, -1);
        Deque<Integer> st = new ArrayDeque<>();
        for (int i = 0; i < nums.length; i++) {
            while (!st.isEmpty() && nums[st.peek()] < nums[i]) {
                ans[st.pop()] = nums[i];
            }
            st.push(i);
        }
        return ans;
    }

    int[] maxSlidingWindow(int[] nums, int k) {
        Deque<Integer> q = new ArrayDeque<>();
        int[] ans = new int[nums.length - k + 1];
        int idx = 0;
        for (int i = 0; i < nums.length; i++) {
            while (!q.isEmpty() && q.peekFirst() <= i - k) q.pollFirst();
            while (!q.isEmpty() && nums[q.peekLast()] <= nums[i]) q.pollLast();
            q.offerLast(i);
            if (i >= k - 1) ans[idx++] = nums[q.peekFirst()];
        }
        return ans;
    }
}
```

<h2 id="q-009">面试问题：如何用三种语言实现归并排序？</h2>

Python：

```python
def merge_sort(nums):
    if len(nums) <= 1:
        return nums
    mid = len(nums) // 2
    left = merge_sort(nums[:mid])
    right = merge_sort(nums[mid:])
    ans = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            ans.append(left[i])
            i += 1
        else:
            ans.append(right[j])
            j += 1
    ans.extend(left[i:])
    ans.extend(right[j:])
    return ans
```

C/C++：

```cpp
#include <vector>
using namespace std;

void mergeSort(vector<int>& a, int l, int r, vector<int>& tmp) {
    if (l >= r) return;
    int m = l + (r - l) / 2;
    mergeSort(a, l, m, tmp);
    mergeSort(a, m + 1, r, tmp);
    int i = l, j = m + 1, k = l;
    while (i <= m && j <= r) tmp[k++] = (a[i] <= a[j]) ? a[i++] : a[j++];
    while (i <= m) tmp[k++] = a[i++];
    while (j <= r) tmp[k++] = a[j++];
    for (int p = l; p <= r; ++p) a[p] = tmp[p];
}
```

<h2 id="q-010">面试问题：如何用三种语言实现二分查找左边界？</h2>

Python：

```python
def lower_bound(nums, target):
    left, right = 0, len(nums)
    while left < right:
        mid = left + (right - left) // 2
        if nums[mid] < target:
            left = mid + 1
        else:
            right = mid
    return left
```

<h2 id="q-011">面试问题：如何用三种语言实现 Top K 小顶堆？</h2>

求最大的 K 个元素时，维护大小为 K 的小顶堆。

Python：

```python
import heapq

def top_k_largest(nums, k):
    heap = []
    for x in nums:
        if len(heap) < k:
            heapq.heappush(heap, x)
        elif x > heap[0]:
            heapq.heapreplace(heap, x)
    return sorted(heap, reverse=True)
```

C/C++：

```cpp
#include <functional>
#include <queue>
#include <vector>
using namespace std;

vector<int> topKLargest(vector<int>& nums, int k) {
    priority_queue<int, vector<int>, greater<int>> pq;
    for (int x : nums) {
        if ((int)pq.size() < k) pq.push(x);
        else if (x > pq.top()) {
            pq.pop();
            pq.push(x);
        }
    }
    vector<int> ans;
    while (!pq.empty()) {
        ans.push_back(pq.top());
        pq.pop();
    }
    return ans;
}
```

Java：

```java
import java.util.*;

class TopK {
    List<Integer> topKLargest(int[] nums, int k) {
        PriorityQueue<Integer> pq = new PriorityQueue<>();
        for (int x : nums) {
            if (pq.size() < k) pq.offer(x);
            else if (x > pq.peek()) {
                pq.poll();
                pq.offer(x);
            }
        }
        return new ArrayList<>(pq);
    }
}
```

<h2 id="q-013">面试问题：如何用三种语言实现二叉树 DFS 与层序遍历？</h2>

Python：

```python
from collections import deque

def preorder(root):
    if not root:
        return []
    return [root.val] + preorder(root.left) + preorder(root.right)

def level_order(root):
    if not root:
        return []
    ans, q = [], deque([root])
    while q:
        level = []
        for _ in range(len(q)):
            node = q.popleft()
            level.append(node.val)
            if node.left:
                q.append(node.left)
            if node.right:
                q.append(node.right)
        ans.append(level)
    return ans
```

C/C++：

```cpp
#include <queue>
#include <utility>
#include <vector>
using namespace std;

struct TreeNode {
    int val;
    TreeNode* left;
    TreeNode* right;
    TreeNode(int v) : val(v), left(nullptr), right(nullptr) {}
};

void preorder(TreeNode* root, vector<int>& ans) {
    if (!root) return;
    ans.push_back(root->val);
    preorder(root->left, ans);
    preorder(root->right, ans);
}

vector<vector<int>> levelOrder(TreeNode* root) {
    vector<vector<int>> ans;
    if (!root) return ans;
    queue<TreeNode*> q;
    q.push(root);
    while (!q.empty()) {
        int sz = q.size();
        vector<int> level;
        while (sz--) {
            TreeNode* node = q.front();
            q.pop();
            level.push_back(node->val);
            if (node->left) q.push(node->left);
            if (node->right) q.push(node->right);
        }
        ans.push_back(level);
    }
    return ans;
}
```

Java：

```java
import java.util.*;

class TreeNode {
    int val;
    TreeNode left, right;
    TreeNode(int val) { this.val = val; }
}

class TreeTraversal {
    void preorder(TreeNode root, List<Integer> ans) {
        if (root == null) return;
        ans.add(root.val);
        preorder(root.left, ans);
        preorder(root.right, ans);
    }

    List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> ans = new ArrayList<>();
        if (root == null) return ans;
        Queue<TreeNode> q = new ArrayDeque<>();
        q.offer(root);
        while (!q.isEmpty()) {
            int size = q.size();
            List<Integer> level = new ArrayList<>();
            for (int i = 0; i < size; i++) {
                TreeNode node = q.poll();
                level.add(node.val);
                if (node.left != null) q.offer(node.left);
                if (node.right != null) q.offer(node.right);
            }
            ans.add(level);
        }
        return ans;
    }
}
```

```python
from collections import deque

def is_bipartite(graph):

            for v in graph[u]:

                    q.append(v)
```

```python
def dijkstra(n, graph, src):
    dist = [float("inf")] * n
    dist[src] = 0
    pq = [(0, src)]

        d, u = heapq.heappop(pq)
        if d != dist[u]:
```

<h2 id="q-025">面试问题：如何用三种语言实现高斯消元？</h2>

```python
def gaussian_elimination(a, b, eps=1e-12):
    n = len(a)
    aug = [row[:] + [b[i]] for i, row in enumerate(a)]
    for col in range(n):
        pivot = max(range(col, n), key=lambda r: abs(aug[r][col]))
        if abs(aug[pivot][col]) < eps:
            raise ValueError("singular matrix")
        aug[col], aug[pivot] = aug[pivot], aug[col]
        for r in range(col + 1, n):
            factor = aug[r][col] / aug[col][col]
            for c in range(col, n + 1):
                aug[r][c] -= factor * aug[col][c]
    x = [0] * n
    for i in range(n - 1, -1, -1):
        s = sum(aug[i][j] * x[j] for j in range(i + 1, n))
        x[i] = (aug[i][n] - s) / aug[i][i]
    return x
```

- 概念题讲定义、复杂度、适用场景；模板题必须能手写代码。
- Python 重表达，C/C++ 重性能和 STL，Java 重工程集合库。
- LRU = 哈希表 + 双向链表。
- 单调栈解决 next greater / smaller，单调队列解决滑动窗口最值。
- 二分左边界统一使用 `[left, right)` 模板更稳定。
- Top K 最大值用大小为 K 的小顶堆。
- 树 DFS 用递归或栈，层序遍历用队列。
- 拓扑排序看入度，Dijkstra 看堆优化松弛。
- KMP 的关键是前缀函数，Trie 的关键是前缀共享。
- 回溯要撤销选择，DP 要定义状态和转移，数值算法要关注误差和稳定性。
