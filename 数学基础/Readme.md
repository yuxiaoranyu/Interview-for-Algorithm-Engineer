## 学习定位

本目录面向 AIGC 算法岗、应用算法岗、大模型工程岗、AI Agent 与多模态方向面试准备，内容在线性代数、微积分、概率统计、最优化基础上，补充了当前大模型、扩散模型、Flow Matching、RLHF/DPO/GRPO、LoRA、量化、长上下文、推理时扩展、多模态生成与检索增强生成中常用、常考、必备的数学知识。

建议阅读顺序：

1. 先掌握线性代数、微积分、概率统计、信息论与优化的基础概念。
2. 再理解 Transformer、扩散模型、Flow/ODE/SDE、对齐学习、低秩微调与量化背后的数学。
3. 接着阅读公式推导篇，重点掌握 Softmax、KL、VAE、Attention、DDPM、DPO、LoRA、量化等高频推导。
4. 然后阅读最新专题，补齐 MLA/GQA/MQA、RoPE Scaling、Speculative Decoding、Test-Time Scaling、MMDiT、RAG 排序融合、PRM/ORM 等新技术背后的数学。
5. 最后用“面试问题”检查自己是否能把公式解释成工程直觉。
