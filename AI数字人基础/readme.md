## 学习定位

本目录面向 AI 数字人、音频驱动视频、虚拟人交互、可控人体生成、3D Avatar 与多模态视频生成方向的算法和工程面试准备。内容整合 2D/3D 数字人、音频驱动、身份保持、虚拟换衣、3D 表示、可微渲染、GAN、Diffusion/DiT、实时交互和工程落地知识。

建议阅读顺序：

1. 先阅读 `01_AIGC时代数字人基础高频考点.md`，建立完整技术地图。
2. 再按专题阅读 `02_可控人体生成与角色动画高频考点.md`、`03_音频驱动数字人与长视频生成高频考点.md`、`04_ID保持换脸与虚拟换衣高频考点.md`，掌握 2D 数字人的主流面试问题。
3. 接着阅读 `05_3D数字人与可微渲染高频考点.md`，系统掌握 3DMM、FLAME、SMPL/SMPL-X、3D 表示、NeRF、3DGS、传统渲染和可微渲染。
4. 然后阅读 `06_GAN数字人与AIGC前沿高频考点.md`，补齐 GAN 时代的 2D/3D 数字人方法，并理解 Diffusion、DiT、Flow、大视频模型带来的最新趋势。
5. 最后结合本地 `数字人` 目录中的开源项目 README 和配置，理解模型落地链路、输入输出约束和部署成本。
